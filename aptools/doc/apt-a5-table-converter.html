Introduction

apt-a5-table-converter is a program that converts a single table in a .txt file to new .a5 file (HDF5 format), or converts a .a5 file containing expected Datasets to new .txt file.

One file is a required input. Inputs can be:

  • calls.txt or calls.a5
  • summary.txt or summary.a5
  • confidences.txt or confidences.a5

Quick Start

Example 1: convert existing .txt to new .a5 
apt2-a5-table-converter \
--txt AxiomGT1.calls.txt \
--a5 AxiomGT1.calls.a5 \
--txt-to-a5
Example 2: convert existing .a5 to new .txt 
apt2-a5-table-converter \
--a5 AxiomGT1.summary.a5 \
--txt AxiomGT1.summary.txt \
--a5-to-txt
