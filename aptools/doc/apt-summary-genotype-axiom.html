Introduction

apt-summary-genotype-axiom is a program that enables genotyping using a summary.a5 or .txt file as input, rather than CEL files

The basic requirements for running apt-summary-genotype-axiom:

  • Summary.txt or summary.a5 file
  • SNP priors model file

Quick Start 

apt-summary-genotype-axiom \
--summary-a5-input-file $INDIR/AxiomGT1.summary.a5 \
--gender-file $INDIR/AxiomGT1.report.txt \
--analysis-files-path $LIBDIR \
--arg-file $LIBDIR/Axiom_PMDA_96orMore_all_cdf.r7.apt-summary-genotype-axiom.mm.SnpSpecificPriors.AxiomGT1.apt2.xml \
--out-dir $OUTDIR \
--log-file $OUTDIR/apt-genotype-axiom.log \
--batch-folder $OUTDIR \
--genotyping-node:snp-posteriors-output true
or, if artifact-reduction-trustcheck-file is available: 
apt-summary-genotype-axiom \
--summary-a5-input-file $INDIR/AxiomGT1.summary.a5 \
--gender-file $INDIR/AxiomGT1.report.txt \
--artifact-reduction-trustcheck-file  $INDIR/AxiomGT1.trustcheck.txt \
--analysis-files-path $LIBDIR \
--arg-file $LIBDIR/Axiom_PMDA_96orMore_all_cdf.r7.apt-summary-genotype-axiom.mm.SnpSpecificPriors.AxiomGT1.apt2.xml \
--out-dir $OUTDIR \
--log-file $OUTDIR/apt-genotype-axiom.log \
--batch-folder $OUTDIR \
--genotyping-node:snp-posteriors-output true
