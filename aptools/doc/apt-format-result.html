Introduction

apt-format-result is an application which allows for the creation of VCF, or PLINK file formats as well as IGV compatible files from Axiom analyzed batches.

Quick Start

For each output type, apt-format-result requires an annotation file and either a --calls-file parameter or a --batch-folder parameter as input.

Creating Text format files

To create Text format files use the commands below: 

    apt-format-result 
	--batch-folder <folder location>  
	--annotation-file <file> 
	--export-txt-file  <file>

Creating VCF format files

To create VCF format files use the commands below: 

    apt-format-result 
	--calls-file <file> 
	--annotation-file <file> 
	--export-vcf-file  <file>

Creating PLINK format files

To create standard PLINK format (sample-based by rows) files use one of the following commands: 

    apt-format-result 
	--calls-file <file> 
	--annotation-file <file> 
	--export-plink-file <file>

To create transposed PLINK format (transposed is snp-based by rows) files use the following commands: 

    apt-format-result 
	--calls-file <file> 
	--annotation-file <file> 
	--export-plinkt-file

Creating IGV Compatible files

 apt-format-result
--batch-folder <batch folder>
--annotation-file <file>
--run-format-igv True
--igv-cndata-file <cnv.a5 file>
--igv-output-prefix <file name>
--igv-export-cnv-baf Default False
--igv-export-cnv-log2ratio Default False
--igv-export-seg-cn Default False
--igv-export-cnv-copynumber Default False
--igv-export-cnv-smooth-signal Default False
--igv-export-seg-loh Default False
--igv-out-dir <output dir location name>

Additional Options

The following optional commands may be added to the command string as desired:

  • snp-list-file: filters output to only the probesets of interest. It is highly recommended to supply this parameter with the Recommended.ps file created by ps-classification engine, which includes only the BestandRecommended=1 probesets from the ps.performance.txt file.
  • snp-identifier-column: allows a column name to be used as SNP identifier for export. This must be one of annotation columns, e.g. {Affy_SNP_ID, dbSNP_RS_ID}. The default value for this is 'probeset_id'
  • pedigree-file: this file can be used if you wish to include basic pedigree information such as family, sample, maternal, ect.
  • export-call-format: sets format of export calls. Available formats: call_code {AA/AB/BB} or base_call {CC/CT/TT} or translated {0/1/2/-1}. Default is call_code. â€ƒ

Usage:

  apt-format-result --batch-folder  --annotation-file  [options]

  or

  apt-format-result --calls-file  --annotation-file  [options]

  or

  apt-format-result --input-plink-file  --export-txt-file  --export-vcf-file  [options]

  or

  apt-format-result [--run-format-cnv true] [cnv-options]

  or

  apt-format-result [--run-format-igv true] [igv-options]
