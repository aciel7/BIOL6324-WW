Introduction

ps-bac tests for batch effects when samples were genotyped in batches (e.g. cases and controls were genotyped separately and together). Batch effects can cause shifts in intensity space, resulting in incorrect cluster fusion when different batch genotype calls are combined. ps-bac tests for a difference in the B-allele freqency in samples that were genotyped separately and then in a larger group with other samples. The output file contains results for each probeset in the pid file: the B-allele frequency in the smaller sample group, the B-allele frequency in the larger sample group, and the test results. "Pass" indicates that there is no significant difference in the B-allele frequencies of a probeset across the two sample groups, and "Fail" indicates that there is a significant difference across the two sample groups. If the test cannot be performed, the output will be NA. Any probesets that have one set of samples that are all NoCall will return a value of NA. Copy-number aware genotype calls and multilallelic genotype calls are not recognized by this function. ps-bac will ignore any copy-number genotype calls or multiallelic genotype calls when performing the test. A warning message will be output for any probeset that has inappropriate genotype calls. More details describing the use of ps-bac can be found in the Axiom Genotyping Solution Data Analysis Guide.

Quick Start

The command run `ps-bac` on on two calls files in the directory `./input` and writes the output to the file BAC_test.txt: ps-bac --call-file-casecontrol ./input/AxiomGT1.calls.txt --call-file-case ./input/AxiomGT1.calls.small.txt --output-dir ./output --bac-intercept 0.12 --bac-file BAC_test.txt

Details

ps-bac output file contains the B-allele frequencies for the full set of batches and the small set of batches, and if a probeset passed or failed the B-allele frequency test. There is one row per probeset. The input files and options for ps-bac are: -__call-file-casecontrol__ - The name of one of the two calls files. -__call-file-case__ - The name of one of the two calls files. -__pid-file__ - The name of a file containing the probeset names to run the B-allele frequency test on. (Optional) -__sample-file__ - The name of a file containing the sample names to use in the B-allele frequency test. (Optional) -__bac-intercept__ - The value to test the BAC differences against: default=0.02.
